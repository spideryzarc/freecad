# FreeCAD macros 

Este projeto tem a finalidade de criar arquivos FreeCAD. Mais especificamente, arquivosde projetos de marcenaria em MDF ou chapas similares com variáveis paramétricas.

## Execução

Em um computador com FreeCAD instalado, execute o arquivo usando o comando:

```bash
freecad.cmd create_panel.py
```

Isso irá gerar um arquivo FCStd na pasta output.
